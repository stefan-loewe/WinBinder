wb_stop_sound

bool wb_stop_sound ([string command])

Stops one or more sounds that were started with wb_play_sound(). Returns TRUE on success or FALSE otherwise.

Parameters

Value of command

Meaning

null, empty or 'all'

Stop all sounds

'wav' or 'wave'

Stop all WAV sounds.

'mid' or 'midi'

Stop all MIDI sounds.
