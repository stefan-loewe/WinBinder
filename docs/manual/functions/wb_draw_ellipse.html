wb_draw_ellipse

bool wb_draw_ellipse (int target, int xpos, int ypos, int width, int height, int color [, bool filled [, int linewidth [, linestyle]]])

Draws a filled or hollow ellipse. The first parameter, target, may be a WinBinder object, a window handle, a drawing surface or a bitmap.

xpos and ypos are the coordinates of the upper-left corner of the bounding rectangle, in pixels. width and height are the dimensions of the ellipse. color is a RGB color value. Set filled to FALSE to draw a border. In this case, linewidth sets the width of the border, in pixels. A linewidth of zero sets the width to 1 pixel.

Returns TRUE on success or FALSE if an error occurs.

Line styles

See function wb_draw_line() for a list of available line styles.
